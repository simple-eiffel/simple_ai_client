Overview

simple_ai_client provides an Eiffel client for AI APIs including Claude and other LLM services. Includes support for vector embeddings and semantic similarity search.

Part of the simple_* ecosystem of focused, single-purpose Eiffel libraries.

Quick Start

Installation

Add to your ECF file:

<library name="simple_ai_client" location="$SIMPLE_AI_CLIENT/simple_ai_client.ecf"/>

Basic Usage

local
    client: AI_CLIENT
    response: STRING
do
    create client.make_with_api_key (api_key)
    response := client.complete ("Explain Design by Contract")
end

Features

  • Claude API integration
  • Vector embedding support
  • Semantic similarity search
  • Environment variable configuration
  • Full Design by Contract support
